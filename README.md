# Algoritmos e Estruturas de Dados — Java

Repositório de estudos dedicado a explorar as principais **estruturas de dados** com implementações práticas em Java, baseado no livro *Estruturas de Dados* de Thiago Leite e Carvalho (Casa do Código / Alura).

___
## Estruturas abordadas

| Estrutura | Descrição rápida |
|---|---|
| **Vetor e Matriz** | Coleções de elementos indexados em uma ou duas dimensões |
| **Pilha** | LIFO — o último a entrar é o primeiro a sair |
| **Fila** | FIFO — o primeiro a entrar é o primeiro a sair |
| **Lista** | Sequência dinâmica de elementos encadeados |
| **Árvore** | Estrutura hierárquica com nó raiz e subárvores |
| **Grafo** | Conjunto de vértices conectados por arestas |
| **Set** | Coleção sem elementos duplicados |
| **HashTable** | Tabela de dispersão para mapeamento chave-valor de alta performance |
 
___
## Como usar este repositório
